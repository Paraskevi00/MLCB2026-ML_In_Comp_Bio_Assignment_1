# MLCB2026-ML_In_Comp_Bio_Assignment_1
The goal of this assignment is to build a complete machine learning pipeline for predicting age using DNA methylation data derived from the GSE40279 dataset. This analysis involves several steps, including: preprocessing the data to handle missing values and ensure consistency, establishing baseline models to assess predictive signal, selecting informative CpG features to reduce dimensionality and improve generalization, and tuning model hyperparameters to achieve optimal performance. Overall, this work aims to demonstrate how machine learning techniques can be effectively applied to high-dimensional biological data to extract meaningful patterns. It also identifies specific features with substantial biological relevance enhancing ongoing clinical research and highlights the challenges of modelling complex biological systems. 
As a complementary task, this study also explores sex prediction from DNA methylation data as a binary classification problem. Unlike age prediction, which relies on distributed and continuous methylation changes, sex prediction is driven by strong, localized signals, particularly from CpG sites associated with X-chromosome inactivation (near the XIST gene). This provides a useful contrast, highlighting how different biological processes require distinct machine learning characteristics and adaptations.
